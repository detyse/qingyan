# 倾言 (QingYan)

> 一个愿意倾听的 AI 伙伴

---

## 项目简介

**倾言** 是一个情绪倾诉与心理支持 AI 项目。

在这个快节奏的世界里，很多人只是需要一个**被听见**的空间。倾言不评判、不说教，只是静静地听你说，然后轻轻地把情绪还给你——也许换了一个角度，也许多了一些光亮。

---

## 核心理念

### 1. 倾听先于解决
不是每个问题都需要答案。有时候，倾诉本身就是疗愈。

### 2. 情绪无垃圾
愤怒、焦虑、悲伤……这些不是"垃圾"，是信号。倾言帮你看见它们背后的需求。

### 3. 解构与重构
把混沌的情绪拆解，再温柔地拼回去——不是改变你，而是帮你看见不同的自己。

### 4. 正向回流
倾诉之后，你会收到一份"美好信息"——可能是另一个视角，一句诗，或只是一个拥抱。

---

## 功能特性

| 功能 | 描述 |
|------|------|
| 🗣️ **自由倾诉** | 想说就说，无需组织语言 |
| 🔍 **情绪识别** | AI 自动识别情绪类型与强度 |
| 🧩 **认知解构** | 拆解情绪背后的思维模式 |
| 💡 **视角转换** | 提供新的看问题角度 |
| 🌱 **正向反馈** | 生成专属的"美好信息" |
| 📓 **情绪日记** | 记录倾诉历史，看见成长轨迹 |

## 使用场景

- **深夜emo** — 睡不着，想找人说说话
- **职场压力** — 被老板骂了，心里堵得慌
- **人际困扰** — 和朋友吵架，不知道怎么办
- **无名焦虑** — 说不清为什么，就是难受
- **日常碎碎念** — 只是想吐槽一下生活

## 快速开始

```bash
# 克隆项目
git clone https://github.com/yourname/qingyan.git
cd qingyan

# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env，填入你的 API Key

# 启动开发服务器
npm run dev
```
